# Joycrew 구글시트 입력 Apps Script

구글시트에서 신규입장자, 언팔필수, 차단필수 목록을 붙여넣고 버튼 한 번으로 명단 시트에 반영하는 스크립트입니다.

## 처음 한 번만 설치

1. 구글시트를 엽니다.
2. 상단 메뉴에서 `확장 프로그램 > Apps Script`를 누릅니다.
3. 기본으로 있는 코드를 모두 지우고 `joycrew_new_entrants.gs` 내용을 전체 복사해서 붙여넣습니다.
4. 저장합니다.
5. 구글시트로 돌아와 새로고침합니다.

## 매주 사용 방법

1. 상단 메뉴에 생긴 `쪼이크루 > 입력/결과 시트 만들기`를 한 번 누릅니다.
2. `신규입장자_입력` 시트의 `A2` 아래에 전달받은 원문을 그대로 붙여넣습니다.
3. `쪼이크루 > 신규입장자 반영`을 누릅니다.
4. 빈 행은 자동 입력되고, 이미 값이 있던 행은 `신규입장자_결과` 시트에 기존 아이디가 표시됩니다.

실수로 `신규입장자_결과` 시트나 현재 보고 있는 시트의 A열에 붙여넣어도, 스크립트가 A2 아래 내용을 한 번 더 찾아서 처리합니다.

## 언팔/차단 사용 방법

1. `언팔차단_입력` 시트의 `A2` 아래에 원문을 붙여넣습니다.
2. 첫 줄이나 내용 안에 `언팔` 또는 `차단`을 넣습니다.
3. `쪼이크루 > 언팔/차단 반영`을 누릅니다.
4. `언팔`은 D열, `차단`은 E열의 빈칸에 자동 입력됩니다.
5. 이미 같은 아이디가 있으면 덮어쓰지 않고 `언팔차단_결과` 시트에 기존 행이 표시됩니다.

예시:

```text
차단
@yunavely_mom
https://www.instagram.com/yunavely_mom
@thx._.babe
https://www.instagram.com/thx._.babe
```

`@아이디`와 `https://www.instagram.com/아이디`가 같이 있어도 같은 아이디는 한 번만 입력합니다.

## 동작 방식

- `9. @a.yun_day` 같은 줄만 읽습니다.
- `@`는 제거하고 `a.yun_day`만 입력합니다.
- 번호 `9`는 시트의 `9행 A열`에 대응합니다.
- A열이 비어 있으면 입력합니다.
- A열에 값이 있으면 덮어쓰지 않고 결과 시트에 표시합니다.
- 언팔/차단은 번호 없이 아이디만 읽고, 각각 D열/E열의 빈칸부터 채웁니다.
